# Rust REST API
A REST API made using the rust language, [diesel](https://crates.io/crates/diesel) ORM and [actix-web](https://crates.io/crates/actix-web) framework

### Build
`cargo build`

### Features
This API contains:
- User authentication using JWT tokens
- REST functionality over all database tables
- Auto-populating database using diesel migrations
- Standalone executable
- Blazing fast performance
- and some more extras...

### Purpose
Its purpose was to serve as the back-end for an example application based on travel experiences. You can infer more about it by reading the [sql creation scripts](./migrations)
